# Employee Salary Prediction Using Machine Learning

## Project Overview
This project predicts employee salaries using Machine Learning with the Linear Regression algorithm. The project includes data preprocessing, exploratory data analysis (EDA), feature encoding, model training, and performance evaluation.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Dataset
The dataset contains employee information such as:
- Age
- Years of Experience
- Education Level
- Job Role
- City Tier
- Performance Score
- Number of Skills
- Remote Work
- Annual Salary (Target)

## Machine Learning Algorithm
- Linear Regression

## Project Workflow
- Import Libraries
- Load Dataset
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Label Encoding
- Train-Test Split
- Model Training
- Salary Prediction
- Model Evaluation

## Model Performance
- MAE: 4137.17
- MSE: 26,398,441.37
- RMSE: 5137.94
- R² Score: 97.99%
